# EasyUtils

## 1.Basic Feature/Tools

- ETaskManager: a simple task manager
- ELog: a log tools

## 2.Install

> use pip or clone source code directly

```
pip install EasyUtils-SPeak
```

or

```
git clone git@github.com:Sunrisepeak/EasyUtils.git
```
